# Recurring Appointments

Set up recurring sessions for clients you see on a regular schedule.

## Creating a Recurring Series

1. Create a new appointment or edit an existing one
2. Toggle **Recurring** on
3. Choose the frequency: **Weekly**, **Biweekly**, or **Custom**
4. Set how many sessions to create
5. Click **Save** — all sessions in the series are created at once

## Managing a Series

When you edit or delete a recurring appointment, you'll be asked:

- **This appointment only** — change just this one session
- **This and future appointments** — change this session and all future ones in the series
- **All appointments** — change every session in the series

## Extending a Series

Need to add more sessions to an existing series?

1. Open any appointment in the series
2. Click **Extend Series**
3. Choose how many additional sessions to add (1–12)
4. The new sessions are added after the last session in the series

## Remaining Sessions

When viewing an appointment in a series, you'll see a badge showing how many sessions remain (e.g., "3 of 12 remaining").
