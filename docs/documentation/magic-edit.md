# Magic Edit

Magic Edit lets you refine any section of your progress note using natural language instructions.

## How to Use Magic Edit

1. After a note is generated, click the **edit** button to enter edit mode
2. Click the **wand icon** (Magic Edit) next to any section header
3. Type what you want changed — for example:
   - "Make the assessment more detailed"
   - "Add that the client reported improved sleep"
   - "Shorten this section"
   - "Rewrite in third person"
4. The AI rewrites that section based on your instruction
5. Review the change and accept or undo

## Example Prompts

| What you want | What to type |
|---------------|-------------|
| More clinical detail | "Expand the assessment with more clinical observations" |
| Add missing info | "Add that we discussed medication adherence" |
| Change tone | "Make this more concise and clinical" |
| Fix an error | "Change 'anxiety' to 'generalized anxiety disorder'" |
| Restructure | "Break this into bullet points" |

## Tips

- Be specific about what you want changed
- You can use Magic Edit as many times as you need — there's no limit
- Each edit only changes the section you selected, not the entire note
- You can always undo a Magic Edit if you don't like the result
