# Couples & Family Therapy

The platform supports couples and family therapy with shared sessions, separate billing, and group notes.

## Creating a Couple or Family Group

1. Schedule a couples or family appointment from the **Calendar**
2. When creating the appointment, select multiple clients — the system will prompt you to create a therapy group
3. Select the group type: **Couple** or **Family**
4. Add existing clients (or create new ones)
5. Designate the **identified patient** and **guarantor** (responsible party for billing)
6. Click **Create Group**

## Scheduling Group Sessions

- When creating an appointment, select the **group** instead of an individual client
- All group members appear on the appointment
- Each member can have separate billing (or bill to the guarantor)

## Group Notes

- When recording a group session, the AI generates a note that captures the group dynamics
- Notes can reference individual and shared goals
- Use the **Couples Therapy** or **Family Therapy** template for best results

## Billing

- **Group billing**: one invoice for the group (charged to guarantor)
- **Individual billing**: separate invoices per member
- Billing accounts are kept separate to maintain proper financial records
