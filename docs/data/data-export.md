# Data Export

Export a client's complete record as a downloadable zip file containing PDFs and original documents.

## What's Exported

- **Patient summary** — demographics and contact information
- **Clinical notes** — all progress notes as PDFs
- **Diagnoses** — active and resolved conditions
- **Treatment plans** — goals, objectives, and interventions
- **Intake forms** — completed questionnaire responses
- **Uploaded documents** — original files (PDFs, images, etc.)
- **Billing summary** — invoice history

## How to Export

1. Go to **Settings > Data Export**
2. Search for and select the **client**
3. Choose what to include (clinical notes, intake forms, documents, billing)
4. Click **Export**
5. The system generates PDFs and bundles everything into a zip file
6. Download the zip file when it's ready

## Export History

Previous exports are listed with:
- Date exported
- File size
- Status (processing, ready, expired)
- Download link (available for 7 days)
