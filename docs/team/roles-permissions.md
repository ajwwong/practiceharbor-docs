# Roles & Permissions

Each team member is assigned a role that determines what they can see and do in the platform.

## Clinician Roles

| Permission | Entire Practice | Full Client List | Basic |
|-----------|:-:|:-:|:-:|
| View all clients | Yes | Yes | No (own only) |
| Edit own clients | Yes | Yes | Yes |
| Edit other clinicians' clients | Yes | No | No |
| View all notes | Yes | Yes | No (own only) |
| Manage settings | Yes | No | No |
| View billing | Yes | Yes | Own clients |

## Administrative Roles

| Permission | Practice Manager | Practice Biller | Practice Scheduler |
|-----------|:-:|:-:|:-:|
| View clients | Yes | Billing only | Schedule only |
| Manage team | Yes | No | No |
| View/manage billing | Yes | Yes | No |
| Manage calendar | Yes | No | Yes |
| Access settings | Yes | No | No |

## Changing a Team Member's Role

1. Go to **Settings → Team**
2. Click the menu icon next to the team member
3. Select **Edit Role**
4. Choose the new role and save
